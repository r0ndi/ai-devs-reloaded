# AI Devs 2 Reloaded
Unofficial CLI for AI Devs

## Configuration and usage
1. Install the dependencies
    ```shell
    npm ci
    ```

2. Copy and configure environment variables
    ```shell
    cp .env.example .env
    ```

3. Start and execute task
    ```shell
    npm start -- taskName
    ```
